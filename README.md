- [GitHub Actions](https://github.com/features/actions) — 自动构建
- [GitHub Pages](https://pages.github.com/) — 静态托管

## 本地开发

```bash
# 安装 Typst
# https://typst.app/open-source/#download

# 构建网站
uv run build.py build

# 本地预览
uv run build.py preview
```
